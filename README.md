Handwritten Digit Recognition using Machine Learning
 📌 Project Overview

This project recognizes handwritten digits using the Scikit-Learn Digits dataset and a Multi-Layer Perceptron (MLP) Classifier. The model is trained to classify handwritten digits from 0 to 9.

 🎯 Objective

* Recognize handwritten digits accurately.
* Learn how to build a Machine Learning classification model.
* Evaluate model performance using accuracy and classification metrics.

🛠 Technologies Used

* Python
* Google Colab
* Scikit-Learn
* NumPy
* Matplotlib

 📂 Dataset

* Scikit-Learn Digits Dataset
* 1797 handwritten digit images
* Image size: 8 × 8 pixels

 🤖 Machine Learning Algorithm

* Multi-Layer Perceptron (MLPClassifier)

 📊 Features

* Load and explore the dataset
* Visualize handwritten digits
* Split data into training and testing sets
* Train an MLP classifier
* Predict handwritten digits
* Evaluate model accuracy

 📈 Results

The trained model successfully recognizes handwritten digits with high accuracy.
